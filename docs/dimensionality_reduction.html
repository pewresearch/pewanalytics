pewanalytics.stats.dimensionality_reduction.correspondence_analysis(edges, n=1)[source]¶

Performs CA on a set of features.

Parameters

  • edges – edges is a dataframe of NxN where both the rows and columns are “nodes” and the values are some sort of closeness or similarity measure (like a cosine similarity matrix)

  • n – Number of dimensions to extract

Returns

A dataframe of the N dimensions
