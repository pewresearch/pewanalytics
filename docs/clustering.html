pewanalytics.stats.clustering.compute_hdbscan_clusters(features, min_cluster_size=100, min_samples=1)[source]¶

Uses HDBSCAN clustering to automatically identify the best number of clusters and map each unit to one. Rows are the units, and columns are features. Returns a series with the cluster labels for each row.

Parameters

  • features – TF-IDF sparse matrix or a Pandas DataFrame

  • min_cluster_size – int - minimum number of documents/units that can exist in a cluster.

  • min_samples – Minimum number of samples to draw (see HDBSCAN documentation for more)

Returns

Series with the cluster ID that each row belongs to

  • k – int, number of clusters

  • return_score – bool, default=False; if True, returns a tuple with the cluster assignments and the silhouette score of the clustering
