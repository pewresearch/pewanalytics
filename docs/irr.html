Computes a variety of inter-rater reliability scores, including Cohen’s kappa, Krippendorf’s alpha, precision, and recall.

Parameters

  • coder_df – A dataframe of codes

  • coder1 – The value in coder_column for rows corresponding to the first coder

  • coder2 – The value in coder_column for rows corresponding to the second coder

  • outcome_column – The column that contains the codes

  • document_column – The column that contains IDs for the documents

  • coder_column – The column containing values that indicate which coder assigned the code

  • weight_column – The column that contains sampling weights

  • pos_label – The value indicating a positive label (optional)
